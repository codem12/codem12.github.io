第一篇(沙箭鱼生小猫)

发表于 2019-11-28 | 分类于 生活 | | 阅读次数:
字数统计: 785 | 阅读时长 ≈ 2

　　人生总会碰到各种事情，尽管印象深刻，但时间可是最强的忘却催化剂，生活中的柴米油盐也好，工作心得也好，兴趣爱好也好，总想记录一下，就有了写博客的念头，嗯，今天开始我的第一篇博客，就记录下生活吧。。。。。。

　　今天，我家的沙箭鱼生小猫了，就结果，19点15分，产出2猫，一黑一虎斑，过程有点小插曲。。。。。。

　　总结一下孕猫生产的注意事项吧。（主要根据这次的经验及网上搜到的各种信息）

  1. 孕猫生产时间，一般从发情第一天开始算，大概66天左右，会有生产预兆，不吃东西，气喘（仔细听），坐立不安，出现这些症状应该临产了，生产是否结束就比较难确定了（产前去做B超检查，明确小猫数量比较好）；
  2. 提前准备好产房（提前2周），产房外结构纸箱就可以了（部分情况，猫猫不喜欢准备好的产房，例如博主家的，把她困到笼子里面，生产的时候自己就会进产房里面），开口方向可以自己发挥，我家如图示，内容底下垫好护理垫，可以放件不用的旧衣服，护理垫可以多准备，新生猫是不会在猫砂盆大小便的，母猫生产结束之后也会恶露，方便清洁；
  3. 接生器械（主要用于剪脐带、吸羊水，淘宝买1套就行），一般情况猫都会自己处理，这是以防万一准备的，博主的老沙是只加菲，首次生产，生产顺利，但不会咬脐带，2只都咬脐带了，但是，有一只没有咬断，只好博主自己动手了，说下过程，准备好工具：酒精、碘伏，手套（橡胶那种，比较好操作，淘宝买一套会配有），剪刀，棉线，镊子（止血钳），开始操作：1.放置好小猫，侧卧位、仰卧位（估计很难），2.距离肚脐2cm左右处，用棉线打结（打结方法见图），用剪刀剪断靠近胎盘这边（规范点可以在1cm处及2cm处打结，剪断两个结之间），3.剪完后，切口用碘伏消毒，4.胎盘可以放孕猫旁边，孕猫会吃。吸羊水没经验，就不写了，注意观察小猫状态，如果不叫不动，可以戴手套抚摸下背部，刺激小猫，注意动作要轻柔，没反应再按网上那套心肺复苏、吸羊水吧。没什么大问题就交给猫猫自己处理，自己在一旁观察就行，不要频繁打扰产猫；
  4. 食物准备：产猫：罐头、幼猫猫粮（容易消化，蛋白质含量高）、水煮鸡肉、羊奶粉，新生猫：羊奶粉，幼猫猫粮（特殊情况特殊对待吧）。看情况喂吧，博主家的产猫生产结束后，尽管饿，也不吃猫粮，喂的水煮鸡肉。

　　哈哈，就这样吧，希望能帮到各位铲屎官。
